#

**Lunar v2** is a feature-packed, modern, browser-inspired tabbed browser-like experience.

---

## Features

- Browser like UI and navigation
- Fast performance
- `about:blank` cloaking
- Built in ad blocking
- Tab system
- Keyboard shortcuts
- And much more

## Deployment via Terminal

> **Note**  
> Before deploying, install:
>
> - [Git](https://git-scm.com/downloads)
> - [Node.js](https://nodejs.org/en/download/prebuilt-installer)
>
> Then install **pnpm**:
>
> ```bash
> npm install -g pnpm
> ```

### Production

1. Clone the repository:

   ```bash
   git clone https://github.com/lunar-proxy/lunar-v2
   cd lunar-v2
   ```

2. Install dependencies and start the server:
   ```bash
   pnpm install
   pnpm start
   ```

### Development

If you are actively developing Lunar v2, use the development server to avoid constant rebuilds:

```bash
pnpm install
pnpm dev
```
